Menggunakan Firebase Authentication.

## Endpoint

- **Sign In**: `FirebaseAuth.instance.signInWithEmailAndPassword`.
- **Sign Out**: `FirebaseAuth.instance.signOut`.

## Contoh Penggunaan

```dart
Future<void> signOut() async {
  await FirebaseAuth.instance.signOut();
```
