---

## 1. Clone Project

Clone dulu repo Maqdis Connect ke laptop kamu:

```bash
git clone [URL_REPO_KAMU]
cd maqdis_connect
```

## **2. Install Dependencies**

Jalankan perintah ini di terminal :

```
flutter pub get
```

---

## **3. Setup Firebase**

- Download file google-services.json (Android) dan GoogleService-Info.plist (iOS) dari Firebase console.
- Taruh file-nya di folder yang sesuai (android/app & ios/Runner).
- Cek juga file .env kalau ada environment variable tambahan.

---

## **4. Run Project**

Jalankan aplikasi di emulator/device:

```
flutter run
```
